POWTOON

Powtoon Ltd. es una empresa británica que vende software de animación basado en la nube (SaaS) para crear presentaciones animadas y videos explicativos animados. El nombre "Powtoon" es una combinación de " PowerPoint " y la palabra " Cartoon ".

Powtoon es un software de animación basado en web que permite a los usuarios crear presentaciones animadas manipulando objetos creados previamente, imágenes importadas, música proporcionada y voces en off creadas por el usuario. Powtoon utiliza un motor Apache Flex para generar un archivo XML que se puede reproducir en el visor en línea de Powtoon, exportar a YouTube o descargar como un archivo MP4.
